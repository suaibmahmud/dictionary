Here,
  main.py and dictionary.kv is attached.
  dictionary using console.py is a different file.

For main.py and dictionary.kv:
1. Create your own project
2. Copy main.py, dictionary.kv and data.json files to that project directory.
3. Run the main.py file

Note:
After creating the project, you have to install some Kivy packages to the interpreter
Go to, File>Setting>Python Interpreter
Click on (+) button
Search, Kivy
Install the below packages:
Kivy
kivy-deps.angle
kivy-deps.glew
kivy-deps.gstreamer
kivy-deps.sdl2

For dictionary using console.py:
Just copy and run the program.
Don't forget to attatch data.json file in the same directory
